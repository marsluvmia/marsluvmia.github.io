## 关于 Hexo

Hexo 是一个快速、简洁且高效的博客框架。它基于 Node.js，支持 Markdown 语法，可以快速生成静态网页。

### 主要特点

- 🚀 **快速**：基于 Node.js，生成速度极快
- 📝 **简单**：使用 Markdown 语法编写文章
- 🎨 **灵活**：支持多种主题和插件
- 📱 **响应式**：适配各种设备屏幕

## 开始写作

你可以使用以下命令创建新的博客文章：

```bash
hexo new "文章标题"
```

然后在 `source/_posts/` 目录下编辑生成的 Markdown 文件即可。
